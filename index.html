Forum website
React, Express.js

My latest project is a full stack forum website built using React and Express.js. The website allows users to create and participate in discussions on various topics. The front-end of the website was built using React, which allowed me to create a responsive and dynamic user interface. The back-end of the website was built using Express.js, which provided an API that simplified the interaction with the Node.js server. The website uses a MySQL database to store user data and discussion topics.
